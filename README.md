# cent-brim

## Fecha
* 2004

## Descripcion del proyecto
* Modificacion de centrifuga brim para trabajar con motor electrico en lugar de motor hidraulico.

## Herramientas utilizadas
* Ninguna

## Modelos de PLC, etc.
* Variador yaskawa
